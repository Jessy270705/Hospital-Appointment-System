# 📊 Student Performance Analysis

## 📄 Description
Analysis of student performance based on various factors like 
study hours, attendance, motivation, and parental involvement.

## 🛠️ Technologies Used
- Python (Jupyter Notebook)
- SQL (MySQL)
- Power BI
- CSV Dataset

## 📁 Files
- `jessy_dataset.csv` - Student dataset
- `jessy.ipynb` - Python analysis notebook
- `jessy.pbix` - Power BI dashboard
- `jessy.sql` - SQL queries

## 📊 Features
- Data cleaning and analysis
- Visual dashboards in Power BI
- SQL queries for filtering students
- Jupyter Notebook for Python analysis
