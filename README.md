## 📁 Project 3: Sentiment Analyzer

**Description:**
An AI-powered sentiment analysis tool that classifies text (positive, negative, neutral) using a pre-trained NLP model from Hugging Face.

**Tech Stack:**
- Python
- Hugging Face Transformers (DistilBERT)
- NLP (Natural Language Processing)

**Features:**
- Analyzes sentiment of any input text
- Uses DistilBERT pre-trained model
- Returns positive / negative / neutral classification
- Fast and accurate results

**Files:**
- `sentiment_analyzer.py` — Main Python script
