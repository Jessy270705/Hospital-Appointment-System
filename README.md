# 🏥 Hospital Appointment System

A web-based hospital appointment booking system that allows 
patients to book, view, and manage their appointments online.

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ✨ Features
- Book a new appointment
- View available doctors and departments
- User-friendly and responsive interface
- Clean and simple design

## 🚀 How to Run
 1. Download or clone the repository
  https://github.com/Jessy270705/Hospital-Appointment-System.git
 2. Open `index.html` in any browser (Chrome/Edge)
 3. No installation needed!
